HTML5

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web. Se estructura mediante etiquetas que definen el contenido y la organización de una página. Las etiquetas básicas incluyen "html", que marca el inicio de un documento, "head" para incluir metadatos, y "body" donde se ubica el contenido visible. Dentro del cuerpo, etiquetas como "h1" se usan para encabezados, "p" para párrafos y "a" para enlaces. HTML es esencial para dar formato, estructura y semántica a las páginas, permitiendo la integración de imágenes, videos y otros elementos multimedia.

CSS3

CSS es el lenguaje utilizado para describir la presentación y el diseño visual de una página web. Controla la apariencia de los elementos HTML mediante propiedades como color, tamaño, márgenes y posicionamiento. CSS permite separar el contenido (HTML) de la presentación, facilitando el mantenimiento y la coherencia en múltiples páginas. Las reglas de CSS se aplican mediante selectores, como clases e identificadores.

JAVA SCRIPT

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web. A diferencia de HTML y CSS, que estructuran y estilizan el contenido, JavaScript permite a los desarrolladores crear funciones dinámicas, como responder a eventos, validar formularios y manipular el DOM. Puede ejecutarse tanto en el lado del cliente como en el servidor. Los principales elementos del lenguaje incluyen variables, funciones y eventos. Además, permite trabajar con objetos y arrays para gestionar datos. Es esencial para el desarrollo web moderno.

PHP

LPHP (Hypertext Preprocessor) es un lenguaje de programación del lado del servidor utilizado para desarrollar aplicaciones web dinámicas. Se integra con HTML y maneja la lógica detrás de las páginas web, como formularios, bases de datos y sesiones de usuario. PHP es ampliamente compatible con servidores web y bases de datos como MySQL. Es popular por su simplicidad y flexibilidad.
